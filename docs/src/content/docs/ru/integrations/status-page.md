---
title: Публичная страница статуса
description: Поделитесь статусом прокси с пользователями
---

Xray Checker включает встроенную публичную страницу статуса, которая позволяет делиться информацией о доступности прокси с пользователями без раскрытия конфиденциальных данных конфигурации.

## Обзор

Публичная страница статуса предоставляет:

- Статус прокси в реальном времени (онлайн/офлайн)
- Информацию о задержке для каждого прокси
- Чистый, адаптивный интерфейс
- Автоматические обновления

При включении публичного режима конфиденциальная информация, такая как адреса серверов, порты и детали конфигурации, скрыты от просмотра.

## Включение публичного режима

Установите переменную окружения:

```bash
WEB_PUBLIC=true
```

Или используйте флаг CLI:

```bash
xray-checker --web-public
```

## Что скрыто в публичном режиме

| Информация            | Публичный режим | Обычный режим |
| --------------------- | --------------- | ------------- |
| Имя прокси            | ✓ Видно         | ✓ Видно       |
| Статус                | ✓ Видно         | ✓ Видно       |
| Задержка              | ✓ Видно         | ✓ Видно       |
| Адрес сервера         | ✗ Скрыто        | ✓ Видно       |
| Порт                  | ✗ Скрыто        | ✓ Видно       |
| Локальный порт прокси | ✗ Скрыто        | ✓ Видно       |
| Детали конфигурации   | ✗ Скрыто        | ✓ Видно       |

## Пользовательское имя подписки

Вы можете установить пользовательское имя для страницы статуса, используя информацию о подписке от вашего провайдера, или установить его вручную:

```bash
# Будет отображаться как заголовок страницы
SUBSCRIPTION_URL=https://provider.com/sub#MyVPN
```

## Настройка URL

Страница статуса поддерживает параметры URL для настройки отображения:

| Параметр                 | Описание                                         |
| ------------------------ | ------------------------------------------------ |
| `hideHeader=true`        | Скрыть шапку с логотипом и элементами управления |
| `hideStats=true`         | Скрыть карточки статистики                       |
| `hideServersHeader=true` | Скрыть заголовок "Серверы"                       |
| `hideControls=true`      | Скрыть поиск и фильтры                           |
| `hideStatusInfo=true`    | Скрыть техническую информацию в подвале          |
| `hideFooter=true`        | Скрыть подвал страницы                           |
| `hideBackground=true`    | Прозрачный фон                                   |

### Пример: Минимальное встраивание

```
https://your-server.com/?hideHeader=true&hideFooter=true&hideControls=true
```

## Сценарии использования

### Для пользователей

Предоставьте пользователям ссылку для проверки доступности прокси:

```
https://status.yourdomain.com/
```

### Встраивание в документацию

Используйте iframe для встраивания страницы статуса:

```html
<iframe
  src="https://status.yourdomain.com/?hideHeader=true&hideFooter=true"
  width="100%"
  height="400"
  frameborder="0"
>
</iframe>
```

### Интеграция со службами статуса

Публичная страница хорошо работает за обратными прокси, такими как Nginx или Cloudflare, для дополнительной безопасности и кэширования.
