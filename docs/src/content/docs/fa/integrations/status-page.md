---
title: صفحه وضعیت عمومی
description: اشتراک‌گذاری وضعیت پروکسی با کاربران
---

Xray Checker شامل یک صفحه وضعیت عمومی داخلی است که به شما امکان می‌دهد اطلاعات دسترسی پروکسی را با کاربران خود به اشتراک بگذارید بدون اینکه جزئیات پیکربندی حساس را فاش کنید.

## بررسی اجمالی

صفحه وضعیت عمومی ارائه می‌دهد:

- وضعیت پروکسی در لحظه (آنلاین/آفلاین)
- اطلاعات تأخیر برای هر پروکسی
- رابط کاربری تمیز و واکنش‌گرا
- به‌روزرسانی خودکار

هنگام فعال بودن، اطلاعات حساس مانند آدرس سرورها، پورت‌ها و جزئیات پیکربندی از دید پنهان می‌شوند.

## فعال‌سازی حالت عمومی

متغیر محیطی را تنظیم کنید:

```bash
WEB_PUBLIC=true
```

یا از فلگ CLI استفاده کنید:

```bash
xray-checker --web-public
```

## چه چیزی در حالت عمومی پنهان است

| اطلاعات          | حالت عمومی    | حالت عادی     |
| ---------------- | ------------- | ------------- |
| نام پروکسی       | ✓ قابل مشاهده | ✓ قابل مشاهده |
| وضعیت            | ✓ قابل مشاهده | ✓ قابل مشاهده |
| تأخیر            | ✓ قابل مشاهده | ✓ قابل مشاهده |
| آدرس سرور        | ✗ پنهان       | ✓ قابل مشاهده |
| پورت             | ✗ پنهان       | ✓ قابل مشاهده |
| پورت پروکسی محلی | ✗ پنهان       | ✓ قابل مشاهده |
| جزئیات پیکربندی  | ✗ پنهان       | ✓ قابل مشاهده |

## نام اشتراک سفارشی

می‌توانید یک نام سفارشی برای صفحه وضعیت خود با استفاده از اطلاعات اشتراک از ارائه‌دهنده تنظیم کنید:

```bash
# به عنوان عنوان صفحه نمایش داده می‌شود
SUBSCRIPTION_URL=https://provider.com/sub#MyVPN
```

## سفارشی‌سازی URL

صفحه وضعیت پارامترهای URL را برای سفارشی‌سازی نمایش پشتیبانی می‌کند:

| پارامتر                  | توضیحات                           |
| ------------------------ | --------------------------------- |
| `hideHeader=true`        | پنهان کردن هدر با لوگو و کنترل‌ها |
| `hideStats=true`         | پنهان کردن کارت‌های آمار          |
| `hideServersHeader=true` | پنهان کردن عنوان "سرورها"         |
| `hideControls=true`      | پنهان کردن جستجو و فیلترها        |
| `hideStatusInfo=true`    | پنهان کردن اطلاعات فنی فوتر       |
| `hideFooter=true`        | پنهان کردن فوتر صفحه              |
| `hideBackground=true`    | پس‌زمینه شفاف                     |

### مثال: جاسازی حداقلی

```
https://your-server.com/?hideHeader=true&hideFooter=true&hideControls=true
```

## موارد استفاده

### اشتراک‌گذاری با کاربران

لینکی برای بررسی دسترسی پروکسی به کاربران خود ارائه دهید:

```
https://status.yourdomain.com/
```

### جاسازی در مستندات

از iframe برای جاسازی صفحه وضعیت استفاده کنید:

```html
<iframe
  src="https://status.yourdomain.com/?hideHeader=true&hideFooter=true"
  width="100%"
  height="400"
  frameborder="0"
>
</iframe>
```

### یکپارچه‌سازی با سرویس‌های وضعیت

صفحه عمومی پشت پروکسی‌های معکوس مانند Nginx یا Cloudflare برای امنیت و کش اضافی به خوبی کار می‌کند.
