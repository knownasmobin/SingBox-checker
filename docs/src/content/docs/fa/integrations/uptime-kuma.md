title: Uptime Kuma
description: گزینه‌ها و مثال‌های استقرار
---

Xray Checker نقاط پایانی وضعیت فردی برای هر پروکسی ارائه می‌دهد، ایده‌آل برای نظارت با Uptime Kuma.

### راه‌اندازی پایه

1. Uptime Kuma را باز کنید
2. روی "Add Monitor" کلیک کنید
3. "HTTP(s)" را انتخاب کنید
4. مانیتور را پیکربندی کنید:
   - نام: نام پروکسی
   - URL: `http://localhost:2112/config/a1b2c3d4e5f67890`
   - فاصله: فاصله بررسی مورد نظر شما
   - تلاش مجدد: ۳ بار توصیه می‌شود

### با احراز هویت

اگر محافظت متریک فعال است:

فرمت URL:

```
http://username:password@localhost:2112/config/a1b2c3d4e5f67890
```

پیکربندی:

- روش احراز هویت: Basic Auth
- نام کاربری: METRICS_USERNAME شما
- رمز عبور: METRICS_PASSWORD شما

### کدهای وضعیت

- ۲۰۰: پروکسی کار می‌کند
- ۵۰۳: پروکسی ناموفق
- ۴۰۱: احراز هویت مورد نیاز است
- ۴۰۳: احراز هویت ناموفق
